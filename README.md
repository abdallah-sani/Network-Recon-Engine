# Network Recon Engine (Automated Port Scanner)

A high-performance network reconnaissance application built to demonstrate the evolution of port scanning techniques, moving from standard socket connections to low-level packet crafting and filtering.

##  Key Features
* **Socket-Based Scanner:** Leverages Python's native `socket` library for quick TCP handshake verification.
* **Npcap Integration:** Utilizes raw packet sniffing and custom filtering engines for deep-packet network auditing.
* **Automated Logging:** Saves structured scan results for security analysis and monitoring.

## 🛡️ Objective
Developed as a foundational portfolio project for defensive cybersecurity applications, network monitoring, and vulnerability assessments.
